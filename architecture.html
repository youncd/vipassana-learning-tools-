아라한 (Arahant)
응공, 모든 번뇌 완전 소멸
현생에서 완전한 해탈
지혜계발로 인한 심리변화
• 이기심 완전히 사라짐 → 이타심 자연스럽게 증장
• 자만심, 욕심, 악행의 뿌리인 아상이 소멸
• 마음이 홀가분해지고 모든 괴로움에서 해방
• 인간으로서의 품격과 수준이 향상
• 성인(聖人)의 경지로 성장
아상 소멸과 사까야 디띨 제거
사까야 디띨 (Sakkāya-diṭṭhi) - 유신견:
• 물질+정신을 '나'로 착각하는 견해
• 걷는 나, 숨쉬는 나, 앉는 나가 실제로는 존재하지 않음
• 복합적 조건들의 집합을 '나'로 착각
• 자동차 비유: 부품들의 집합체일 뿐 '자동차'는 없음

아상이 깨지는 순간:
• 아상·인상·중생상·수자상 모두 소멸
• 남자도 여자도 너도 나도 원래 없었음을 깨달음

Nāma-rūpa-pariccheda-ñāṇa
첫 번째 위빠사나 지혜:
• 물질은 물질, 정신은 정신 - 완전히 별개
• 기름과 물, 물과 불처럼 전혀 섞이지 않음
• 정신: 인지기능 있음 (살아있는 사람)
• 물질: 인지기능 없음 (시체, 나무토막)

매 순간 관찰:
• 들숨(물질) ↔ 들이쉬는 마음(정신)
• 날숨(물질) ↔ 내쉬는 마음(정신)
• 1초에 1번씩, 1시간에 3600번 관찰
개념 초월의 3단계
1. 나마 빤냐띨 (Nāma-paññatti)
이름 개념 초월: 단어, 명칭을 넘어서기
'오른발 걷다', '들숨 날숨' 등의 언어적 개념

2. 산타나 빤냐띨 (Santāna-paññatti)
모양 개념 초월: 형태, 모습을 넘어서기
다리모양, 배모양, 걸음모양 등의 시각적 개념

3. 사무하 빤냐띨 (Samuha-paññatti)
집합 개념 초월: 복합체를 해체하여 보기
섞여있는 지수화풍과 마음상태들을 각각 분리
빠라마타 (Paramattha) - 궁극적 실재
개념을 넘어선 진실:
• 모든 개념이 깨어진 후 보이는 실제
• 1/4초, 1/8초, 1/16초 단위로 관찰
• 깐나(시간) 덩어리가 완전히 해체됨
• 위빠사나 지혜 = 특별한 봄, 인사이트

구체적인 수행 실천법
앉기 수행:
• 앉아있는 몸 (물질적 실재) + 앉으려는 마음 (정신적 실재)
• 엉덩이와 의자가 닿는 팽팽한 힘
• 자세를 지키는 뻣뻣한 힘들
• 매순간 물질과 정신을 분리해서 관찰

걷기 수행:
• 오른발 걷다 → 걷고자 하는 마음 + 걷는 물질
• 다리가 들리고 앞으로 나가서 내려가는 과정
• 물질적 과정과 정신적 과정의 상호작용

호흡 수행:
• 들숨 (풍대) + 들이쉴려는 마음
• 날숨 (풍대) + 내쉴려는 마음
• 매순간 이 두 가지만 굴러가고 있음을 관찰

갈애(渴愛) 생성 과정
좋은 느낌 → 탐심 (탐):
욕심나게 되어 더 많이 갖고 싶어함
갑자기 죽으면 귀신으로 태어남

나쁜 느낌 → 성냄 (진):
싫어하며 그 상황에서 벗어나고 싶어함
갑자기 죽으면 지옥에 태어남

덤덤한 느낌 → 어리석음 (치):
멍때리고 무명상태
갑자기 죽으면 축생으로 태어남
36 차까 수따 (Cha-chakka Sutta)
6×6=36 갈애:
• 보면서 갈애 (안식 + 색경)
• 들으면서 갈애 (이식 + 성경)
• 냄새맡으면서 갈애 (비식 + 향경)
• 맛보면서 갈애 (설식 + 미경)
• 감촉하면서 갈애 (신식 + 촉경)
• 생각하면서 갈애 (의식 + 법경)

이것은 과보로서 피할 수 없는 자연스러운 현상
육촉(六觸)과 육수(六受)
육촉 (Ṣaḍ-sparśa)
감각기관+대상+의식이
만나는 접촉 순간
안촉, 이촉, 비촉, 설촉, 신촉, 의촉
육수 (Ṣaḍ-vedanā)
접촉에서 생기는
여섯 가지 느낌
좋은 느낌, 안 좋은 느낌, 덤덤한 느낌
육입-육경-육식 시스템 상세
육입 (六入)
Ṣaḍ-āyatana

1. 안(眼) - 눈
2. 이(耳) - 귀
3. 비(鼻) - 코
4. 설(舌) - 혀
5. 신(身) - 몸
6. 의(意) - 마음
육경 (六境)
Ṣaḍ-viṣaya

1. 색(色) - 형상
2. 성(聲) - 소리
3. 향(香) - 냄새
4. 미(味) - 맛
5. 촉(觸) - 감촉
6. 법(法) - 법경
육식 (六識)
Ṣaḍ-vijñāna

1. 안식 - 보는 의식
2. 이식 - 듣는 의식
3. 비식 - 냄새 의식
4. 설식 - 맛 의식
5. 신식 - 감촉 의식
6. 의식 - 생각 의식
십이연기법 (Dvādaśa-nidāna) 상세
무명(無明) → 행(行) → 식(識) → 명색(名色) → 육입(六入) → 촉(觸) → 수(受) → 애(愛) → 취(取) → 유(有) → 생(生) → 노사(老死)

Avidyā → Saṃskāra → Vijñāna → Nāma-rūpa → Ṣaḍ-āyatana → Sparśa → Vedanā → Tṛṣṇā → Upādāna → Bhava → Jāti → Jarā-maraṇa

우리의 존재:
식으로 태어나고 → 명색으로 물질정신이 있고 → 육입으로 여섯 감각기관이 있고 → 색성향미촉법과 마주치게 되고 → 안이비설신의라는 보고 듣고 냄새맡고 맛보고 감촉하고 생각하는 마음이 벌어진다
업(業)과 과보(果報)
과보: 우리가 했던 업으로 받게 되는 것
태어났기 때문에 육입이 있고, 육경이 있고, 촉하고, 느끼고,
안이비설신의 마음 따라 행복하고 괴롭고 덤덤하다

자연스러운 반응:
좋으면 욕심나고, 나쁘면 싫어하고, 덤덤하면 어리석게 반응
이것이 불선업으로 이어져 사악처에 갈 수 있다
